# Claude Code Plugins

Claude Code plugins for AI agent development.

## Plugins

### openai-agents
OpenAI Agent SDK plugin for building AI agents with tools, handoffs, guardrails, and sessions.

### google-adk
Google Agent Development Kit plugin for building AI agents with multi-agent systems, callbacks, and deployment.

## Installation

```bash
# Install individual plugins
/plugin install openai-agents@postlabs/claude-code-plugin
/plugin install google-adk@postlabs/claude-code-plugin
```
