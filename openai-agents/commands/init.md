# /openai-agent:init

Initialize a new OpenAI Agent SDK project.

## Usage

```
/openai-agent:init [project-name]
```

## What it does

1. Creates project structure
2. Sets up virtual environment
3. Installs openai-agents SDK
4. Creates basic agent template

## Project Structure

```
project-name/
├── agents/
│   └── main_agent.py
├── tools/
├── pyproject.toml
├── requirements.txt
└── README.md
```
