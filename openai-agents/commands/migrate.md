# /openai-agent:migrate

Migrate to new OpenAI Agent SDK versions.

## Usage

```
/openai-agent:migrate [target-version]
```

## Migration Process

1. Check current SDK version
2. Review breaking changes
3. Update code for compatibility
4. Test functionality

## Examples

```
/openai-agent:migrate 0.3.0
/openai-agent:migrate latest
```
