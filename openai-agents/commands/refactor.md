# /openai-agent:refactor

Refactor agent code for better structure and maintainability.

## Usage

```
/openai-agent:refactor [request]
```

## Common Refactoring Tasks

- Split large agents into specialized agents
- Extract reusable tools
- Improve prompt instructions
- Add type hints
- Organize code structure

## Examples

```
/openai-agent:refactor split this agent into router + specialists
/openai-agent:refactor extract common tools to shared module
/openai-agent:refactor improve code organization
```
