# /openai-agent:dev

General development command for OpenAI Agent SDK projects.

## Usage

```
/openai-agent:dev [request]
```

## Examples

```
/openai-agent:dev add error handling to my agent
/openai-agent:dev improve the prompt instructions
/openai-agent:dev add logging
```

## Behavior

Uses the openai-agents-sdk skill to understand the SDK and provide contextual assistance.
