# /google-adk:refactor

Refactor agent code for better structure and maintainability.

## Usage

```
/google-adk:refactor [request]
```

## Common Refactoring Tasks

- Split large agents into specialized agents
- Convert to multi-agent architecture
- Extract reusable tools
- Improve agent instructions
- Add type hints
- Organize code structure

## Examples

```
/google-adk:refactor convert to multi-agent system
/google-adk:refactor extract common tools
/google-adk:refactor improve code organization
```
