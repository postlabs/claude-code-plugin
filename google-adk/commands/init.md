# /google-adk:init

Initialize a new Google ADK project.

## Usage

```
/google-adk:init [project-name]
```

## What it does

1. Creates project structure
2. Sets up virtual environment
3. Installs google-adk SDK
4. Creates basic agent template

## Project Structure

```
project-name/
├── agents/
│   └── main_agent.py
├── tools/
├── tests/
├── pyproject.toml
├── requirements.txt
└── README.md
```
